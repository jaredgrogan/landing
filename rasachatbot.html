Introduction to Rasa

Rasa is an open-source machine learning framework for automated text and voice-based conversations.

Setup

pip install rasa rasa init

Natural Language Understanding

Configure your NLU model in the nlu.yml file:

nlu: - intent: greet examples: | - hey - hello - hi - hello there

Dialogue Management

Define your conversation flows in the stories.yml file:

stories: - story: greet user steps: - intent: greet - action: utter_greet

Training the Bot

Train your Rasa model:

rasa train

Exercises

  1. Create a new intent and add examples to the nlu.yml file.
  2. Expand your stories in the stories.yml file to include more conversation paths.
  3. Test your bot using the Rasa shell.
  4. Deploy your bot using Rasa X.
